# SPACE INAVDER

Space Invader Game is developed in Cocos Creator 3.6 (Typescript).

## Player Movemnt

To move Player ,touch the screen to start and move player using arrow keys in keyboard.

### Bullet fire

To Fire bullet using Space bar in your keyboard.
